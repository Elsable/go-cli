# Golang CLI Task manager

A sample cli application in golang using cobra and boltdb.

# 

### Add task

```sh
$ task add Fetch water
$ task add Visit grocery
$ task add Call mama
```

### List task

```sh

$ task list

You have the following tasks
1. Fetch Water
2. Visit grocery
3. Call mama
```

### Do Tasks

```sh
$ task do 1

Marked "1" as completed.
```
